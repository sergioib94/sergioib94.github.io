* Añade un nuevo modelo al fichero polls/models.py:

~~~
  class Categoria(models.Model):	
  	Abr = models.CharField(max_length=4)
  	Nombre = models.CharField(max_length=50)

  	def __str__(self):
  		return self.Abr+" - "+self.Nombre 		
~~~
